## 选项

        'path' => '',

        'path_view' => 'view',
视图路径

        'path_view_override' => '',
用于覆盖的路径——用于插件模式

关闭  View 视图的 notice 警告，以避免麻烦的处理。

## 详解

DuckPhp\Core\View 的选项共享一个 path,带一个 path_view.

path_view 如果是 / 开始的，会忽略 path 选项

当你想把视图目录 放入 app 目录的时候，请自行调整 path_view
