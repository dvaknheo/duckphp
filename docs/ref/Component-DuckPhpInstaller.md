## 简介

安装器组件

## 选项
全部选项

        'path' => '',
安装路径

        'namespace' => '',
安装的命名空间

        'force' => false,
安装器，强制安装，覆盖现有文件

        'autoloader' => 'vendor/autoload.php',
安装器，自动加载器指向位置

        'verbose' => false,
安装器，显示详情

        'help' => false,
安装器，显示帮助

## 说明

DuckPhpInstaller 是辅助安装类。

`选项`同时作为`命令行参数`使用

一般不在系统里加载，使用以下命令查看帮助

```
vendor/bin/duckphp new  --help
```
